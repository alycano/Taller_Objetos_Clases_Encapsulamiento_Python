En este primer taller me enfoqué en entender cómo funcionan las clases y los objetos en Python mediante un ejercicio práctico de una biblioteca. Creé una clase llamada Libro que guarda la información básica como el título, el autor y las páginas, además de una variable para saber si el libro se puede prestar o no. Programé la lógica para que el programa avise cuando un libro cambia de estado al ser prestado o devuelto, evitando que se preste dos veces lo mismo, y añadí una función para imprimir toda esa información junta y ver cómo se actualizan los datos al ejecutar el código.
