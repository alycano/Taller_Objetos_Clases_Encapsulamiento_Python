En este repositorio comparto la solución a los dos talleres de Programación Orientada a Objetos que trabajamos en clase. En el primer taller, aprendí a crear una base con clases y objetos usando un ejemplo de una biblioteca para gestionar préstamos de libros. En el segundo taller, subí el nivel usando encapsulamiento para crear una cuenta bancaria, donde protegí los datos para que el saldo no se pudiera cambiar por error y solo aceptara valores válidos. Como conclusión, estos ejercicios me sirvieron para entender que programar con objetos ayuda mucho a que el código sea más organizado y que, al proteger la información con atributos privados, el sistema se vuelve mucho más seguro y difícil de romper cuando otros lo usan.
